题目名称
诗兄去旅行

描述
世界那么大，诗兄想去看看。

师兄想旅行散散心，众所周知诗兄是个勤俭持家的男孩纸，面对地图，他想找出到目的地最短的距离。

地图是这样的，每个地图包含编号为（1 ~ N）的 N 个城市，且符合以下规则：
1. 城市之间的道路是单向的，所以如果 城市A 能通向 城市B，那么 城市B 不能通向 城市A。
2. 已知到每个城市只能通向比它编号大的城市。
3. 每个城市最多能通往另外两个城市。
4. 城市之间不存在环。

输入
输入包含多组数据，以 EOF 结束。
第一行包含一个整数 N ( 2 <= N <= 20)，代表城市的数量。
接下来是 N 行，包含三个整数 M X Y ( 1 <= M, X, Y <= 20)。M 为城市的编号，X 与 Y 表示 M 城市能通往 X 城市 与 Y 城市。当 X 或 Y 为 0 时，代表不通向任何城市。例如 "A B 0" 代表 城市A 只能通向 城市B，"A 0 0" 代表城市A 不能通向任何城市。
所有输入都是合法的。

输出
输出从 城市1 前往 城市N 经过最少的城市数。

输入样例
4
1 2 3
2 0 0
3 0 4
4 0 0
5
1 2 3
2 4 0
3 0 5
4 5 0
5 0 0

输出样例
3
3
