题目名称
诗兄的分形

描述
最近诗兄迷上了研究分形，要知道分形最大的一个特点是：无论生成多大的分形，它们之间总是自相似的。
诗兄很想知道分形变大后是怎样的。这是一个庞大的问题，诗兄最近忙着征婚，所以他找到了会编程的你。

分形的形状请参考样例。

输入
正整数 n (1 <= n <= 6)

输出
输出边长为 3^n 的分形，分形的增长方法参考样例。

Case 1:
1

Case 2:
2

Case 3:
